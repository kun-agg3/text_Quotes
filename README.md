# text_Quotes

## General Info
```
Created an application to send a motivational quote to your phone everyday at a specific time.

What I have right now is a demo but,it can be modified for greater use cases.

Scrapped a website full of quotes
```
## Technologies
Project is created with:
* TWILIO
* PUPPETEER

